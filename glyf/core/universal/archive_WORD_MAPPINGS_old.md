## Usage Examples

```python
from glyphabetic import decode_vector

# Decode a flowing growth signature
result = decode_vector([0.8, 0.2, 0.1, 0.2, 0.7, 0.3, 0.2])
print(result['description'])
# → "flow through growth (like TREE)"

# Encode from glyphs
from glyphabetic import encode_word
vec = encode_word("C+S")  # Flow + Growth
print(vec.to_list())
# → [0.7, 0.0, 0.0, 0.0, 0.7, 0.0, 0.0]
```
