### What Was Built

**MainActivity.kt hardened with:**
1. **Triple-layer resurrection**
   - Bundle (fastest) — `onSaveInstanceState/Restore`
   - Disk (survives kill) — `cryogenize/resurrect` to `filesDir`
   - Timing instrumentation — logs exact ms via `System.nanoTime()`

2. **Choreographer frame sync**
   - Replaced coarse `delay(50)` with true 60 FPS `FrameCallback`
   - Janky frame detection (>16.67ms × 1.5 threshold)
   - Live FPS and κ (fidelity) metrics

3. **Performance overlay**
   - Resurrection source (bundle/disk/none)
   - Resurrection time in ms
   - FPS counter, jank ratio, κ metric
   - Energy state and harmonic H0

4. **SD persistence**
   - `LatticeState.cryogenize(File)` — writes 96 bytes
   - `LatticeState.resurrect(File)` — reads 96 bytes
   - Triggered on every `onSaveInstanceState`
