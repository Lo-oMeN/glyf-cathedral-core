### 2. GLM TOKENIZER PIPELINE ANALYSIS

Pipeline: Phonetic → Glyphiform → Primitive → Geometric Relative → Geometric Universal

**Phonetic Layer Findings:**
- Vowel systems show triangular/conical geometric constraints (three-vowel systems: /i, a, u/ form acoustical triangle)
- Consonant places of articulation map to spatial coordinates (labial → coronal → velar as spatial progression)
- Universal CV syllable preference reflects rhythmic geometric patterning

**Glyphiform Layer Findings:**
- Visual Word Form Area (VWFA) demonstrates letter recognition invariant to:
  - Case (upper/lower)
  - Font type
  - Size
  - Location on retina
  - Color
- This supports the existence of **abstract geometric primitives** in letter recognition
- fMRI studies (Cohen et al. 2000-2025) show consistent VWFA activation across individuals
- 2025 research confirms VWFA location is individual-specific but task-agnostic

**Primitive Layer Findings:**
- 7 basic geometric primitives identified across writing systems:
  1. Dot/circle (zero, vowel markers, periods)
  2. Vertical line (I, 1, |)
  3. Horizontal line (—, 一, −)
  4. Diagonal/slanted (/, \\, <, >)
  5. Curve (C, S, ~)
  6. Angle/corner (L, 7, Γ)
  7. Crossing/intersection (+, ×, 十)

**Geometric Relative Layer Findings:**
- African fractal geometry (Eglash 1999) demonstrates recursive self-similarity as a cultural invariant
- Bamana sand divination uses recursive Cantor-set-like construction
- Yoruba adire cloth folding creates dragon-curve patterns
- Quincunx pattern (5 squares) appears across West African cultures
- These patterns are not "accidental" but explicit algorithmic constructions

**Geometric Universal Layer Findings:**
- Golden ratio (φ = 1.618...) emerges in:
  - Fibonacci sequences in human gait patterns (Iosa et al. 2021, 2025)
  - EEG phase-locking values (Hohwy & Seth 2023)
  - Autistic cognition optimization (a:e = φ ratio hypothesis)
  - Brain morphology and connectivity networks (Zhang et al. 2022)
- Circle/ring forms appear in:
  - Nearly all writing systems' enclosing marks

### 4. CROSS-CULTURAL GEOMETRIC MAPPINGS

**African Fractals (Eglash 1999):**
- Settlement architecture: recursive rectangular spirals (Logone-Birni palace)
- Textiles: recursive scaling in kente cloth, adire patterns
- Art: iterative construction in Mauritanian leatherwork
- Religion: quincunx pattern represents "light of Allah"
- Games: owari/mancala as one-dimensional cellular automaton
- Divination: Bamana sand divination uses recursive Cantor-like sets

**Universal Geometric Constraints:**
1. **Closure**: All writing systems develop enclosing marks (circles, boxes)
2. **Linearity**: Text flows in linear sequences (left-right, right-left, top-bottom)
3. **Differentiation**: Minimal contrast pairs (short/long, curved/straight)
4. **Repetition**: Iterative patterns for emphasis, rhythm, decoration
5. **Scaling**: Same forms at different sizes (fractal aesthetic)

**Cultural Variation Within Constraints:**
- Directionality varies (L→R, R→L, T→B) but linearity is universal
- Specific characters vary but primitive shapes are shared
- Decorative complexity varies but recursive principles are widespread
